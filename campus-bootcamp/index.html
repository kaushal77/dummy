A Weekend Bootcamp to uplift the interest of Coding, Development and Analytics for College Students


Programming Pre-Bootcamp

Pre-requisites:

Want to start coding from fresh, and have basic knowledge of C/C++ syntax (if/else, functions, arrays, loops)

Curriculum:

Introducing to Online Coding Platforms using Array-based problems, Structures, Pointers, Linked List, STL (stacks, queues, vector, map, pair, set)

Apply
Algorithms Pre-Bootcamp

Pre-requisites:

In-depth Implementation and Problem-solving using LinkedList, Stacks, Queues, Trees plus basic questions of Dynamic Programming and Graphs

Curriculum:

Advanced Trees, Dynamic Programming & Graph Theory

Apply
Python Pre-Bootcamp

Pre-requisites:

A mindset to learn and explore the power of python and an intention to code plus a step to Machine Learning or Scripting

Python Syntax, Python Data Structures, Python Functional, E-Commerce website Scraping program

Apply
Analyst Pre-Bootcamp

Pre-requisites:

A mindset to brainstorm on case studies, business problems and tools required for data and business analysis

Curriculum:

Guestimation, Case study, Introduction to Analytical tools like SQL & Excel. Tips and tricks for standing out in Group Discussions.

Apply
Machine Learning Pre-Bootcamp

Pre-requisites:

Python Programming (can code Python programs using loops, function, strings, list, dictionaries and file system ), basic understanding of Machine Learning / Data Science & love towards Mathematics.

Curriculum:

Python libraries for Data Cleaning, Data Visualisation, Data Modeling, Mathematics & Statistics. Implementing and discussing Kaggle problems.

Apply
Deep Learning Pre-Bootcamp

Pre-requisites:

Python syntax & libraries like Numpy, Pandas, Sklearn, Matplotlib, Statistics & Probability, Linear Algebra, Calculus, Machine Learning Algorithms, Dimensionality Reduction Technique (PCA), Accuracy Measurement Techniques.

Curriculum:

Artificial Neural Network, Tensorflow Basics, Convolutional Neural Network and Kaggle problem using Deep Learning & Tensorflow

Apply
System Design Pre-Bootcamp

Pre-requisites:

Backend Programming in any of the tech stack, Knowledge of servers, Linux and GIT.

Curriculum:

Distributed Systems, System Design, Dockers, Orchestration using Cloud and corresponding concepts like message queues, data pipeline, etc

Apply
Full-stack Pre-Bootcamp

Pre-requisites:

Basics of HTML & CSS and mindset to build Web Application from scratch

Curriculum:

JavaScript, HTML / CSS, Browser-Based Application, Awareness to Backend Programming majorly REST APIs.
